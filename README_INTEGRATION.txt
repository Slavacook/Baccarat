ЧТО ДАЛЬШЕ:
1. Запустите Godot
2. Нажмите F5 для обновления файловой системы
3. Откройте Game.tscn
4. Запустите игру (F5)
5. Протестируйте:
   - Включение toggles → фишки появляются
   - Раздача карт
   - Выбор победителя
   - Клик на фишку → PayoutScene

ВОЗМОЖНЫЕ ОШИБКИ:
- Если Godot не видит менеджеры → перезапустите Godot
- Если узлы не найдены → проверьте имена узлов в Game.tscn
- Если PayoutScene не открывается → проверьте путь к сцене
